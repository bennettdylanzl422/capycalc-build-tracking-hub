## What It Does

CapyCalc serves as a single-page browser utility for CapybaraGo players who want a quicker way to manage builds, stats, and in-game reference material. Rather than juggling separate planning tools, it combines calculators, lookup data, and tracking views into one interface for faster choices.

Its main purpose is hands-on planning support: you can keep character sheets organized, compare specialization routes, estimate damage, and monitor event or activity plans in one place. The app is meant for players who want a lightweight browser solution for reviewing game data and shaping their next setup.
